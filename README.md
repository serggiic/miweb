# miweb

Repositorio público que contiene una web estática (portfolio) construida con HTML y CSS.

Arquitectura mínima:
- `index.html` — página principal
- `styles.css` — estilos
- `img/` — imágenes y recursos estáticos
- `LICENSE` — licencia del proyecto

El contenido de este repositorio es la web en sí; no incluye instrucciones de despliegue ni procesos operativos.
